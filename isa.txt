Material da ISA (Os Braços)
Ação: Isa vai mudar os braços para algo mais "forte".

1. Criar a branch develop localmente e rastrear ela remotamente
git switch -c develop origin/develop

2. Criar e Publicar
git switch -c feat/bracos
git push -u origin feat/bracos
